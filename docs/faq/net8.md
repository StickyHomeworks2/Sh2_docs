出现这种弹窗一般是==没安装 .NET 8 运行时==

**解决方法：**

::: steps

1. 点击 `Download it now` 下载 .NET 运行时

2. 运行下载的安装程序安装 .NET 运行时

3. 安装完成后重新打开本软件即可

:::
