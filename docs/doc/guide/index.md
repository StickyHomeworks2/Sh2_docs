## 欢迎使用 StickyHomework2 文档页面
StickyHomeworks2 是一款支持富文本的桌面端作业贴工具，旨在帮助学生和教师更高效地管理和查看作业任务。  
通过简洁直观的界面设计，用户可以轻松添加、编辑和组织作业内容，提高学习和教学的效率。

::: important
StickyHomeworks 项目由 [HelloWRC](https://github.com/HelloWRC) 创建并实现绝大部分功能，现由 [Jizilin](https://github.com/jizilin6732) 继续开发并更名为 StickyHomeworks2。  
StickyHomeworks2 是一个开源项目，现托管在 [GitHub 仓库](https://github.com/StickyHomeworks2/StickyHomeworks2)，采用 [General Public License v3](https://github.com/StickyHomeworks2/StickyHomeworks2/blob/master/LICENSE.txt) 许可协议发布。
:::
