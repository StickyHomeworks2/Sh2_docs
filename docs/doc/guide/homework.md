- 在主界面点击**右上角的添加按钮（+）** 或点击 **下方的布置按钮** 添加作业

  ![添加作业](/images/creat.png)

- 在弹出的编辑面板选择科目并编辑文本内容和格式

  ![编辑作业](/images/main.png)

## 编辑面板组成
- **科目**
- **截至日期**
  - 到指定日期后自动删除当前作业
- **标签**
  - 可以设置在设置里添加的标签
- **文本设置区**
  - ~~表情~~（待实现）
    - 添加 emoji 表情
  - ~~图片~~（待实现）
    - 添加图片来补充说明
  - ~~链接~~（待实现）
    - 添加链接来补充说明
  - 快捷工具栏
    - 撤回
    - 复制
    - 剪贴
    - 粘贴
  - 颜色
    - 为文本设置颜色
  - 去除颜色
  - 快捷颜色
    - 添加后可快速为文本添加颜色
